# Agtran Test Frontend

This project build from VUE JS

how to project Setup is here :
## Project setup

```curl
yarn install
```

### Compiles and hot-reloads for development

```curl
yarn serve
```

### Compiles and minifies for production

```curl
yarn build
```

### Lints and fixes files

```curl
yarn lint
```
